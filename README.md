# City_Ride-PowerBI_Dashboard

Interactive Power BI dashboard analyzing ride booking data including revenue trends, vehicle performance, cancellations and customer ratings.

## Project Overview

This project presents an interactive Power BI dashboard analyzing city ride booking data.  
The dashboard provides insights into booking trends, revenue performance, vehicle usage, cancellations, and customer ratings.

The goal of this project is to demonstrate how data visualization can help businesses understand ride demand patterns and improve operational decisions.

## Dashboard Features

The dashboard includes multiple report pages designed to provide different perspectives of the ride booking data.

* Overall Dashboard – Summary of bookings and key KPIs
* Vehicle Performance – Booking distribution and Distance Travelled by vehicle type  
* Revenue Analysis – Revenue trends, payment method and top customers insights
* Cancellation Analysis – Driver and customer cancellation patterns with Reasons
* Rating Insights – Customer feedback and service ratings with Peak Hour
* Summary Report – Overall performance overview with Important slicer.

## Key Performance Indicators (KPIs)

The dashboard tracks important business metrics including:

* Total Bookings  
* Total Revenue  
* Average Ride Value  
* Cancellation Rate  
* Customer Rating Distribution  
* Vehicle Type Performance
* Peak Hour Rate
  
## Tools & Technologies

Power BI – Data visualization and dashboard creation  
DAX – Data calculations and measures  
Data Cleaning – Preparing the dataset for analysis

The dataset contains ride booking records including:

* Booking ID
